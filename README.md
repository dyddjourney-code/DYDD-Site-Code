# Discover Your Divine Design Site Code

This repository stores the current Discover Your Divine Design website page code in an organized structure.

## What is here

- `raw/DYDD_Site_Code.txt` is the original uploaded note, preserved as received.
- `fragments/` contains one Squarespace-ready HTML snippet per page.
- `pages/` contains simple browser preview wrappers for each page snippet.
- `docs/site-map.md` documents the page map and navigation structure.
- `assets/css/site.css` is only for local preview/navigation around the snippets.

## Notes

The uploaded note is Squarespace section code, not a full standalone app. The files in `fragments/` are the safest source for copy/paste back into Squarespace. The files in `pages/` are lightweight previews so we can inspect and improve each section locally.
