# FruitLife 360 PDFMonkey Template

This folder contains the first PDFMonkey HTML template for the FruitLife 360 report.

## Files

- `fruitlife-360-report.html` - paste this into a new PDFMonkey template.

## Template Setup

Create a new PDFMonkey template named:

`FruitLife 360 Report`

Use the payload row from the Google Sheet tab:

`PDFMonkey_Payload_360`

The template currently uses the standard Discover Your Divine Design logo as a placeholder:

`https://images.squarespace-cdn.com/content/v1/685da500fbad741e29c08c78/156f49e9-3121-4dd4-8aa6-ac23f67962b7/DYDD-logo.png?format=500w`

Replace that logo URL when the FruitLife-specific logo is ready.

## Notes

- The tone intentionally avoids shame, deficiency language, and holiness grading.
- The report uses the DYDD green/sage visual system with fruit-specific accent colors.
- The report is mapped to the flat `PDFMonkey_Payload_360` fields currently produced by the live Google Sheet.
- The DesignID section is a placeholder lens until the FruitLife + DesignID interpretation library is built.
