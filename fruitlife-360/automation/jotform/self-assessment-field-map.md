# FruitLife 360 Self Assessment Jotform Field Map

Purpose: starts a FruitLife 360 assessment, creates the participant record, and captures the self response.

## Core Fields

- `participant_name` - Full name
- `participant_email` - Email
- `participant_id` - Hidden field generated by Make, or created from submission ID if Make is first step
- `assessment_mode` - Hidden or dropdown, default `FruitLife 360`
- `review_type` - Hidden, value `Self`
- `reviewer_group` - Hidden, value `Self`
- `reviewer_name` - Hidden or copied from participant name
- `reviewer_email` - Hidden or copied from participant email
- `relationship` - Hidden, value `Self`
- `designid_status` - Dropdown: Known, Not sure, Not taken
- `designid_primary` - Dropdown: Architect, Artisan, Shepherd, Steward, Not sure
- `designid_secondary` - Dropdown: Architect, Artisan, Shepherd, Steward, None, Not sure

## Consistency Rating Fields

Use a 1-5 scale.

Scale language:

- 1 = Rarely visible
- 2 = Occasionally visible
- 3 = Somewhat steady
- 4 = Usually steady
- 5 = Deeply consistent

- `Patience_Pressure`

## Written Response

- `Final_Reflection`

Prompt:

Which fruit do you most desire the Holy Spirit to grow in you during this season?

## Make Mapping Into `Response_Intake`

- `Timestamp` from Jotform submission time
- `Participant_ID` from generated ID
- `Participant_Name` from `participant_name`
- `Review_Type` = `Self`
- `Reviewer_Name` from participant name
- `Reviewer_Email` from participant email
- `Relationship` = `Self`
- `Reviewer_Group` = `Self`
- response ID and invite token generated in Make
- consistency, pressure, ranking, and reflection fields map by exact key
