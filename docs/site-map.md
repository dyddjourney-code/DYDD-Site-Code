## Repository Structure

- `raw/` keeps the original uploaded note unchanged.
- `fragments/` keeps each Squarespace-ready page section as a standalone HTML snippet.
- `pages/` wraps each fragment in a simple preview page.
- `assets/css/` contains preview-only repo styling.

## Page Files

- `journey`: Journey — Six-step DYDD journey page: Identity, Expertise, Story, Desire, Spiritual Gifts, and Niche.
- `services`: Services — Ways to engage the DYDD journey through live, online, and resource-based pathways.
